📌 Project Description

The Multispeciality Hospital Management System is a web-based application developed using Java, JSP, Servlets, JDBC, MySQL, Apache Tomcat, HTML, CSS, and JavaScript. It provides separate modules for Admin, Receptionist, Doctor, and Patient to simplify hospital management through role-based access.

Project Workflow

👤 Patient

* Register and Login
* Book an appointment with a preferred doctor and date.
* Receive notifications when the appointment is Confirmed, Rejected, or Rescheduled.
* View and download the prescription as a PDF.

👩‍💼 Receptionist

* Receive a notification whenever a patient books an appointment.
* View all appointment requests.
* Confirm, Reject, or Reschedule appointments.
* Send appointment status notifications to patients.
* Notify the assigned doctor when an appointment is confirmed.

👨‍⚕️ Doctor

* Receive notifications for confirmed appointments.
* View assigned appointments.
* Add diagnosis, medicines, dosage instructions, and remarks.
* Save prescriptions for patients.

👨‍💼 Admin

* Add and manage Departments.
* Add and manage Doctors.
* Add and manage Receptionists.
